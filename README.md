# Портфолио профессии "ML-инженер"
Данные проекты были выполнены в ходе трудовой деятельности, в роли freelanser, а так же участия в хакатонах и различных соревнованиях. Проекты размещены в порядке убывания, в начале наиболее свежий, в конце самый поздний. Предлагаю ознакомиться с исследовательским анализом и первоначальным прототипом, так как дальнейшая часть разработки попадает под NDA.
